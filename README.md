# Tarun Bayya Portfolio

Static portfolio website for `tarunbayya.com`.

## Files

- `index.html` - profile homepage
- `projects.html` - selected projects and work samples
- `contact.html` - contact, resume, education, certifications
- `styles.css` - shared styling
- `script.js` - active navigation and footer year
- `documents/` - downloadable resume and selected work samples
- `CNAME` - custom domain configuration for GitHub Pages

## Deployment Notes

This site is ready for static hosting through GitHub Pages, Netlify, Vercel, Cloudflare Pages, or any standard web host. To make `tarunbayya.com` accessible publicly, the domain DNS must point to the chosen host.
